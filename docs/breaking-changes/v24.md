# Breaking changes in v24

- Bumps `eslint-plugin-unicorn` from v56 to v65
  - v57 went ESM-only, and this repo did not support it until recently.
  - Rather than add breaking changes for multiple intermediate bumps, we jump 8 major versions, to avoid churn in config.
