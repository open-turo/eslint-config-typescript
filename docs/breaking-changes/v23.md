# Breaking changes in v23

- Removes the **`@open-turo/eslint-config-typescript/recommended`** entry point and **`.eslintrc`-style** `recommended.cjs`. This package is **flat config only** (`eslint.config.*`). ESLint v10 no longer supports legacy configuration; migrate using the [ESLint migration guide](https://eslint.org/docs/latest/use/configure/migration-guide) if you still use `.eslintrc`.
- Replaces `eslint-plugin-import` with `eslint-plugin-import-x` for ESLint v10 support (and performance improvements)
- Drops support for ESLint v8
